# Rexcap _(0.1.0 alpa)_
Fullstack Multiplatform App with React + Express + Capacitor
_(this template is still under development)_
## Installation
Clone this repo
```bash
npm install / npm i
# then
npm run dev (web)
# or
npx cap add android
npm run android
*(Live Reload still cannot be used)* 
```
